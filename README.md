# Cloud Resume Challenge + AWS DevOps Portfolio

## Overview

This project demonstrates hands-on experience with AWS Cloud, DevOps, Infrastructure as Code, CI/CD, Containers, and Serverless Architecture.

## Architecture

User → CloudFront → S3 Static Website

User → API Gateway → Lambda → DynamoDB

GitHub → GitHub Actions → AWS Deployment

Docker → ECR → ECS Fargate

## Technologies

* AWS S3
* CloudFront
* API Gateway
* Lambda
* DynamoDB
* ECS Fargate
* ECR
* IAM
* Terraform
* Docker
* GitHub Actions

## Features

* Static Resume Website Hosting
* Visitor Counter using Lambda and DynamoDB
* HTTPS via CloudFront
* CI/CD using GitHub Actions
* Containerized Flask Application
* Infrastructure as Code using Terraform
